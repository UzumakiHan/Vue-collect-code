# Vue-collect-code
二维码相册小程序（Vue3版本）
# 创作背景：
## 22年出入各场所都要提供场所码，以及核酸需要提供个人的核酸码，每次现场扫描比较耗时间，所有创作该小程序用来保存一些工作生活中常用的场所码和核酸码，以及一些图片也可
## 功能主要要上传、编辑、删除、预览跳转
